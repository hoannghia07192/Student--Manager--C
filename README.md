# Student Study Helper (Trợ lý học tập sinh viên)

Đây là dự án cá nhân đầu tay của mình viết bằng ngôn ngữ C. Ứng dụng giúp sinh viên quản lý điểm số, tính toán khả năng rớt môn và quản lý deadline, TO-DO-LIST.

## Tính năng chính
* **Quản lý môn học:** Thêm, Sửa điểm, Xóa môn học.
* **Cảnh báo rớt môn:** Tự động tính toán điểm thi cần đạt để qua môn dựa trên điểm quá trình.
* **To-Do List thông minh:** Quản lý bài tập và tự động cảnh báo khi deadline còn dưới 3 ngày.
* **Lưu trữ dữ liệu:** Tự động lưu và khôi phục dữ liệu từ file text.

## Các Kỹ thuật sử dụng
* Ngôn ngữ: C
* Kỹ thuật: Struct, Pointer, File I/O, Multi-file organization.

## Cách cài đặt và chạy
1. Clone dự án:
   `https://github.com/hoannghia07192/Student--Manager--C.git`
2. Biên dịch (sử dụng GCC):
   `gcc main.c monhoc.c file.c -o app`
3. Chạy chương trình:
   `./app`
